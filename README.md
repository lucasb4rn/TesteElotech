## Pessoa
   -  GET pessoa/{idPessoa}
   -  GET pessoa/buscaTodasPessoas
   -  GET pessoa/buscaTodasPessoas?page=0&size=3&sort=nome //PAGINAVEL
   - POST pessoa/adicionaPessoa
   - PUT pessoa/atualizar
   - DELETE pessoa/deletaPessoa/{idPessoa}
